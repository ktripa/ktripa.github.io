How SPI is computed here

# per grid cell, per calendar month (McKee et al. 1993, Thom 1958 gamma fit)
P_k   = rolling_sum(precip, k)                 # k-month accumulation
A     = ln(mean(P)) - mean(ln(P))              # Thom statistic
alpha = (1 + sqrt(1 + 4A/3)) / (4A);  beta = mean(P)/alpha
H(x)  = q + (1-q) * GammaCDF(x; alpha, beta)   # q = P(zero precip)
SPI   = NormInv(H(x))                          # standard normal transform
